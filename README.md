### Projects

### [Blockchain-based Intelligent Disbursement in NSP](https://github.com/LifnaJos/Blockchain-based-National-Scholarship-Portal) - (August 2023 - November 2024)

- Members: Neelkanth Khithani, [Kushl Alve](https://www.linkedin.com/in/kushl-alve/), [Vedang Gambhire](https://www.linkedin.com/in/vedang-gambhire-114049254/), Shivam Choubey, [Atharva Hande](https://www.linkedin.com/in/atharva-hande/), [Asmi Rajbhar](https://www.linkedin.com/in/asmi-rajbhar-818613239/)
- Abstract: The [National Scholarship Portal (NSP)](https://scholarships.gov.in/) faces challenges like non-automated systems, complex processes, physical verifications, fraud, delays, and data security. This project proposes a blockchain-based solution to create a tamper-proof, streamlined, and secure system, enhancing trust and accountability in scholarship management. [[Paper published in IET Blockchain - November 2024](https://ietresearch.onlinelibrary.wiley.com/doi/10.1049/blc2.12092)]

### [Cloud-based Project Security Pipeline](https://github.com/Neelkanth-khithani/Cloud-based-Project-Security-Pipeline) - (February 2025 - April 2025)

- Members: Neelkanth Khithani, [Ishwari Dehadray](https://www.linkedin.com/in/ishwari-dehadray-684498264/), [Atharv Inamdar](https://www.linkedin.com/in/atharv-inamdar-149247285)
- Abstract: An automated security pipeline using GitLab CI/CD for Static Application Security Testing (SAST).
